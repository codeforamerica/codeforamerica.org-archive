About

Considering applying for the Code for America fellowship, but have some questions you’d like answered? Lingering doubts? Things you’d like clarified? Fret no more: the fellows are here to help! We held a live Google Hangout with current CfA fellows on Tuesday, July 23. Fellows were available to speak about our experiences, and answer any questions about the program and application process. And don't forget to apply by July 31!
