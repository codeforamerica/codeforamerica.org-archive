About

What does the process of opening data actually entail? Join us as we discuss the "how" of open data. We'll start with the big picture process of opening data, and discuss some of the most common roadblocks. We'll unpack frequently used terminology and place it in the context the the larger problem of publishing data sets.

This event is hosted by the Code for America Peer Network, CfA's community of innovators working in local government from over thirty cities and counties. Learn more about the Peer Network and how you can get involved here.
