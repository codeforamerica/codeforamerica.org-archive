About

This webinar will focus on communicating useful knowledge and context for sketch planning and other planning initiatives within transit. The topics covered will include basic philosophies in transit planning, common considerations that can be anticipated when planning for and kicking off a transit project, and helpful transit planning data resources. The Transitmix team will provide demonstrations throughout the presentation on how their app can help local governments and communities work together to achieve transit planning goals.

This event is hosted by the Code for America Peer Network, CfA's community of innovators working in local government from over thirty cities and counties. Learn more about the Peer Network and how you can get involved here.
