Citizens have the power to help their cities.

Here are some simple ways for you to get started with civic hacking.

Have an hour? Code for America Right Now

These are open GitHub Issues labeled with help wanted from across the civic technology movement. Find more using the Civic Tech Issue Finder.

Have one night a week? Go to your local volunteer Brigade hack night.

Have a year for public service? Apply to be a Code for America Fellow

Want a career change? There has never been a better time to consider working in government.
